팀 프로젝트

# Skills

- JAVA
- JSP/Servlet
- Javascript/jQuery
- CSS/HTML
- Oracle

# Menu

- 일반 회원 서비스  
  ```
  회원가입/로그인/탈퇴, 회원 정보 관리, 아이디/비밀번호 찾기(이메일 전송), 도서 목록 조회/검색, 
  감상평/감상태그, 도서 장바구니/결제, 주문 조회
  ```

- 출판사 회원 서비스
  ```
  도서 관리
  ```

- 관리자 회원 서비스
  ```
  회원 정보 관리, 전체 도서 목록 관리
  ```
